# 技术域（domain）

`<domain>` 定义代码所依赖的外部技术环境（操作系统、第三方库、运行时环境），是四维坐标中标识技术边界与可移植性的维度。技术来源。

## 取值
### 基础库

|技术域 | 备注         ||
|-------|-----------------|------|
| none  | 零依赖，连标准库都没用||
| std   | 语言标准库       |   |
| posix | POSIX 接口      |   |
| win32 | Windows API     |   |
| linux | Linux 特有      |   |
